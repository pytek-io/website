## Take advantage of the most advanced web technology using solely Python.

* Allows you to easily write web aps as simple Python scripts.
* Designed from the bottom up to transform the development and operation of business applications.
*  Allows you to develop custom-built solutions adapted to your needs effortlessly and quickly.

### High level and familiar paradigm

Reflect relies on a powerful dependency engine, similar to Excel but much more general. Its interface is heavily inspired from [MobX](https://mobx.js.org/README.html) with a few significant extra functionalities. It allows developers define complex applications behaviour in a natural way.

### Deploy and operate applications seamlessly

Reflect relies on a very robust yet flexible application server. It is a lightening fast server connecting web clients to python processes. It features a builtin support for continous integration and deployment.
