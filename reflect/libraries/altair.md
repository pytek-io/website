[Altair](https://altair-viz.github.io/) is a declarative visualization library for Python based on Vega and Vega-Lite. Thanks to the fact that it relies on the powerful yet simple Vega-Lite visualization it provides an elegant way to to produce beautiful visualizations. Altair API can be used out of the box by wrapping an Altair `Chart` into a Reflect `Chart`.
